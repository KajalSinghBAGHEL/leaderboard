Report an issue

If you are seeing any data related issue on Leaderboard, please raise a ticket on Keka about it.

Follow the steps below to tell us about the issue you have encountered.

1. Go to Keka.
2. In the "Me" section, go to Helpdesk/My Tickets.
3. Raise a New Ticket.
4. Select category as "Leaderboard Data", and describe your issue. We will get back to you.
